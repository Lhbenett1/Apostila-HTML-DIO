Ferramentas Utilizadas

Para começarmos a desenvolver com o HTML, iremos precisar de duas ferramentas bem básicas, no caso de nosso curso, iremos utilizar um editor de textos no caso de nosso curso, o Visual Studio Code, onde iremos adicionas a extensão: "Live Server" que ira ser utilizado para que seu código consiga ser atualizado automaticamente após cada mudança realizada nele, fazendo não necessitar ficar atualizando a página a cada mudança, e um navegador, que será utilizado para rodar o seu código.

Inspetor de Elementos

O inspetor de elementos é uma ferramenta que todo navegador possui e é muito útil para verficiar como a estrura de HTML e CSS daquela página foi criada, podendo até mesmo sabermos como que cada parte especifica da estrutura foi criada. Para isso basta apertar a tecla F12 no caso dos navegadores Edge e google.

Estrutura Básica HTML

Ao dar um ! + enter em uma página recém criada em HTML, toda a sua estrutura dó site será criada.
As partes mais importantes da estrutura de seu site são:

  • <head> Demarcar a área a qual adicionaremos as configurações da página que não ficaram visiveis ao úsuario
  • <html lang:"" "> Definir ao seu navegador em que lingua o seu site está escrito.
  • <meta charset="UTF-8" Definir que modelo de codificação os caracteres apresentados pelo site estão, que no caso é o UTF-8
  • <title> Definir qual que será o nome da aba daquela página.
  • <body> Demarcar a área a qual adicionaremos o cónteudo da página, que será mostrado para o úsuario.

Falando sobre Tags

Tag's são utilizdas para demarcar a estrutura de seu site, além de adicionar conteúdo em seu site, podendo ser apenas abertas como por exemplo a tag <input> ou abertas e fechadas como por exemplo a tag <strong>

Atributos das Tag's

Atributos em tag's são utilizados para adicionar propriedades a uma Tag, sendo alguns atributos obrigatorios a ter, como por exemplo a tag <img> com o atributo "src" que é obrigatorio ter para selecionar a imagem que será adicionada em seu site, e algumas são opcionais, como por exemplo o atributo "id" que servira para selecionar aquela tag em especifico.

Textos

Os textos em seu site são criados apartir de títulos os quais são criados com as tag's de <H1> a <H6> sendo o <H1> o título com a maior importancia e o <H6> com a menor importância, e de páragrafos que são criados utilizando a tag <P>

Além disso os textos possuem tag's semanticas, que servem para dar um sentido a mais para quele texto, como por exemplo as tag's:

  • <Blockquote:> Serve para realizar uma citação. Exemplo:
    Uma citação
  • <Strong:> Serve para deixar o texto em negíto. Exeplo: Negrito
  • <Mark>: Serve para deixar aquela parte do texto marcado, como se fosse grifada por um grifa texto. Exemplo: Grifada
  • <Sup>: Deixar a parte marcada mais para cima, que nem os números elevados a pótencia. X2
  • <Sub>: Deixar o caractere marcado marcado para baixo, igual a formulas químicas. Exemplo: H20

Listas Ordenadas e Não Ordenadas

Listas em HTML são bastante utilizadas, não apenas para a criação de listas propriamente ditas, mas também para a criação de menus de navegação. Existem três tipos de listas em HTML: ordenadas, não ordenadas e listas mistas.

Ordenadas

As listas ordenadas são aquelas que devem seguir uma ordem específica, por isso seus itens são representados com um número na frente.

Para a criação dessas listas devemos utilizar a tag <ol> para demarcar a área que iremos utilizar para a criação de listas ordenadas, e dentro da <ol>, iremos utilizar a tag <li> para adicionarmos os itens dentro da lista.

Exemplo:

<ol>
<li>Item 1<li>
<li>Item 2<li>
<li>Item 3<li>
<ol>

Ficando assim:

  1. Item 1
  2. Item 2
  3. Item 3

Não Ordenadas

As listas não ordenadas são aquelas que não seguem uma ordem específica, por isso seus itens são representados apenas com uma bolinha na frente.

Para a criação dessas listas devemos utilizar a tag <ul> para demarcar a área que iremos utilizar para a criação de não listas ordenadas, e dentro da <ul>, iremos utilizar a tag <li> para adicionarmos os itens dentro da lista.

Exemplo:

<ul>
<li>Item 1<li>
<li>Item 2<li>
<li>Item 3<li>
<ul>

Ficando assim:

  • Item 1
  • Item 2
  • Item 3

Links

Links são utilizados para conectar sites, criar ligações entre diferentes páginas e navegar entre seções da própria página.

Conectar sites

Para fazer a ligação entre um outro site, é bastante simples, basta envelopar uma palavra com a tag <a> e no atributo "href: " basta adicionar o link do site que deseja que o úsuario possa acessar.

Além disso o atributo "target" dentro da tag <a>, serve para que possamos selecionar como que o novo link será aberto, onde as duas opções mais utilizadas são o "target = "_blank" que faz com que o link seja aberto em uma nova aba, e o "target = _self" para que o site seja aberto na mesma aba que o seu site está.

Outro atributo interessante que podemos adicionar dentro da tag <a> é o title, que serve para quando o úsuario deixar o mouse em cima do link aparecer um pequeno balão com o texto referindo para onde que o link o levará.

Podemos utilizar a ligação entre diferentes páginas de seu mesmo site, para isso basta envelopar uma palavra com a tag <a> e no atributo "href: " devemos pressionar ctrl + espaço e selecionar a página de seu site que você desja ir.
