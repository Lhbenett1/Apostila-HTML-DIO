Ferramentas Utilizadas

Para começarmos a desenvolver com o HTML, iremos precisar de duas ferramentas bem básicas, no caso de nosso curso, iremos utilizar um editor de textos no caso de nosso curso, o Visual Studio Code, onde iremos adicionas a extensão: "Live Server" que ira ser utilizado para que seu código consiga ser atualizado automaticamente após cada mudança realizada nele, fazendo não necessitar ficar atualizando a página a cada mudança, e um navegador, que será utilizado para rodar o seu código.

Inspetor de Elementos

O inspetor de elementos é uma ferramenta que todo navegador possui e é muito útil para verficiar como a estrura de HTML e CSS daquela página foi criada, podendo até mesmo sabermos como que cada parte especifica da estrutura foi criada. Para isso basta apertar a tecla F12 no caso dos navegadores Edge e google.

Estrutura Básica HTML

Ao dar um ! + enter em uma página recém criada em HTML, toda a sua estrutura dó site será criada.
As partes mais importantes da estrutura de seu site são:

  • <head> Demarcar a área a qual adicionaremos as configurações da página que não ficaram visiveis ao úsuario
  • <html lang:"" "> Definir ao seu navegador em que lingua o seu site está escrito.
  • <meta charset="UTF-8" Definir que modelo de codificação os caracteres apresentados pelo site estão, que no caso é o UTF-8
  • <title> Definir qual que será o nome da aba daquela página.
  • <body> Demarcar a área a qual adicionaremos o cónteudo da página, que será mostrado para o úsuario.
